### PyInstaller参数
```bash

  --onefile \                    # 单文件模式
  --windowed \                   # 无控制台窗口
  --name OmegaDownloadTool \     # 程序名称
  --add-data "config.json;." \   # 包含配置文件

  download_tool.py               # 源文件
```

#### 1. 程序无法启动
- **检查系统兼容性**: 确保Windows 7+

- **服务器状态**: 确认服务器正常运行
